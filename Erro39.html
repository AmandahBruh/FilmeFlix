O erro 504 gateway timeout ocorre quando um servidor não consegue carregar uma página que solicita recursos de outro servidor. Basicamente, ele acontece quando um servidor tenta carregar uma página web, mas não tem resposta de outro servidor que precisa acessar para completar o carregamento da página.
